# NammaCounsellor Client

Frontend portal for the KCET Predictor and Counselling Companion. Built using modern UI/UX design patterns, TailwindCSS, TypeScript, and React.

## Dev Server Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build production bundle:
   ```bash
   npm run build
   ```
