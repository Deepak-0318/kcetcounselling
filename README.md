# NammaCounsellor

A MERN-based platform that helps KCET students find colleges and branches based on:

- KCET Rank
- Category
- Round
- Branch Preference
- Rank Range

## Features

- **Search & Predict**: Get dynamic college recommendations based on your rank and specific seat categories.
- **College Comparison**: Compare two colleges side-by-side on branch availability and rounds.
- **My Shortlist**: Save your preferred options categorized into Dream, Target, and Safe.
- **Counselling Analytics**: Analyze branch popularities and average cutoffs.

## Tech Stack

- **Frontend**: React + TypeScript + Vite + TailwindCSS
- **Backend**: Node.js + Express.js
- **Database**: JSON Dataset (Optimized static cutoff records)
